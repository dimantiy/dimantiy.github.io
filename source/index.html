Пробы пера

Комбинатор

Небольшая программа для поиска всех возможных комбинаций чисел, сумма которых равна заранее заданному числу (по умолчанию 10). Остановился на решении только для положительных чисел и соответсвенно положительной суммы.

Маркерная доска

Эксперимент с canvas. Приложение реализовано на подобии электронных аналогов маркерных досок, на которых могут размещаться различные элементы: стикеры, текстовые блоки, изображения. Из функциональных возможностей есть загрузка информации о доске с сервера, zoom & map, перемещение элементов по доске. Помимо этого в ней реализовано два механизма рендеринга (Canvas и DOM) с возможностью динамического переключения между ними.

Струна

Реализация механизма воспроизведения звука колеблющейся струны. Движение струны расчитывается на основе численного решения уравнения колебания математической струны (конечно-разностная схема). Считывание звукового сингнала со струны осуществляется с помощью Быстрого Преобразования Фурье.

Быки и коровы

Реализаци небезызвестной игры "Быки и коровы", которая включает в себя как сторону, загадывающую число, так и отгадывающую.
